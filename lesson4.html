CSS 規則組成

CSS 由 選擇器 (selector) 和 宣告區塊 (declaration) 組成。

selector {
   property: value;
   property2: value2;
}

CSS 套用方式

  • 行內樣式 Inline：寫在標籤內
<p style="font-size:16px">文字內容</p>
  • 內嵌樣式 Embedded：寫在 <style> 內
<style>
p { background-color: red; }
</style>
  • 外部樣式 External：連結 CSS
<link rel="stylesheet" href="style.css">
  • 匯入樣式 Import
<style>
@import url("style.css");
</style>

媒體類別（Media Types）

不同裝置可使用不同 CSS。

<link rel="stylesheet" href="style.css" media="print">
<style media="screen"> ... </style>

CSS 選擇器

1. 群組選擇器

h1, h2, h3 { color: green; }

2. 後代選擇器

li strong {
   font-style: italic;
}

3. Class 選擇器

.center { text-align: center; }

4. ID 選擇器

#red { color: red; }
#green { color: green; }

5. 屬性選擇器

[title] { color: red; }
[title="Hello"] { border: 5px solid blue; }

CSS 樣式：背景

背景顏色：

p { background-color: gray; }

背景圖片：

p { background-image: url(bg.jpg); }

背景重複：

body {
   background-image: url(pic.png);
   background-repeat: repeat-y;
}

背景定位：

body {
   background-position: center top;
}

CSS 盒模型（Box Model）

每個元素包含：

  • 內容 content
  • 內距 padding
  • 邊框 border
  • 外距 margin

CSS 定位（Position）

  • static
  • relative
  • absolute
  • fixed
  • sticky
