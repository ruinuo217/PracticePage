HTML 基本文件結構

<!DOCTYPE html>
<html lang="zh-Hant">
<head>
    <title>Hello World</title>
</head>

<body>
    <p>Hello World!</p>
</body>
</html>

HTML 檔頭資訊 <head>

常見標籤：

  • <title>：網頁標題
  • <meta charset="utf-8">：宣告編碼
  • <meta name="keywords" content="HTML, CSS">
  • <meta name="description" content="網站描述">
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">

HTML 頁面內容 <body>

常用 style 屬性：

background-color: 顏色;
background-image: url(圖片位置);

margin-left: 左距;
margin-top: 上距;
margin-right: 右距;
margin-bottom: 下距;

style 行內樣式

  • 文字對齊：text-align
  • 字型：font-family
  • 字體大小：font-size
  • 文字顏色：color
  • 強制換行：word-wrap: break-word;

圖片文繞圖：

  • float: left / right
  • clear: both

標題、段落與列表

<h1>這是標題一</h1>
<h2>這是標題二</h2>
<h3>這是標題三</h3>

<p>這是一個文字段落</p>

<ol>
    <li>項目一</li>
    <li>項目二</li>
</ol>

<ul>
    <li>清單項目</li>
</ul>

水平線 <hr>

<p>段落 1</p>
<hr>
<p>段落 2</p>

HTML 圖片語法

<img src="url" width="寬" height="高">

img 為「空標籤」，不需要結束標籤。

HTML 超連結

<a href="網址">連結文字</a>

可連到：其他網站、本地網頁、圖片、頁面錨點等。

超連結屬性

1. href 屬性：指定連到哪裡

可連到網頁、檔案、圖片、email、錨點等。

<a href="https://google.com">Google</a>

2. id 屬性：設定錨點（頁面內跳轉）

<h2 id="section2">第二段</h2>
<a href="#section2">跳到第二段</a>

3. target 屬性：決定在哪裡開啟連結

  • _blank — 在新視窗開啟
  • new — 在新視窗開啟（類似 _blank）
  • _top — 在最上層框架開啟
  • _parent — 在父層框架開啟
  • _self — 在原頁面開啟（預設）

範例：

<a href="https://google.com" target="_blank">新視窗開啟 Google</a>
