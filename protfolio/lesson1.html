重點內容

🔹 網頁的組成

  • HTML — 建立內容結構
  • CSS — 控制樣式與排版
  • JavaScript — 處理互動效果
  • Server Script（如 PHP）— 處理伺服器邏輯
  • Database（如 MySQL）— 資料儲存與管理

🔹 PHP（伺服端語言）

PHP 是一種伺服端程式語言，用來製作動態網頁。 使用者開啟網站時，伺服器會先執行 PHP 程式碼，再把結果回傳給瀏覽器。 它是開源、跨平台，廣泛用於各類網站的後端開發。

🔹 MySQL（資料庫）

MySQL 是一套常見的資料庫管理系統（DBMS）， 用於整理、管理與儲存大量網站資料。 由於其開源、快速與穩定，廣泛應用於中小型與大型網站。

🔹 開發工具

  • Visual Studio Code（VS Code）— 最常用的前端開發工具
